The Intel Berkeley Research Lab dataset contains readings from 54 sensors deployed across a lab environment. Each sensor (mote) reports temperature, humidity, light, and voltage at approximately 31-second intervals. We use only the temperature modality to study classification performance as a function of feature dimensionality.

  4. Pivot to epoch × sensor matrix
  5. Forward-fill gaps ≤ 3 consecutive NaN epochs
  6. Drop epochs missing any of the 5 required sensors

Label Construction

Binary labels are created by splitting on the median temperature of the reference sensor R (mote 22): class 0 if T ≤ 22.30°C, class 1 if T > 22.30°C. This yields approximately balanced classes (50/50 by construction).

Sensor Selection Criteria

Sensors are selected automatically based on data quality and inter-sensor correlation structure. The goal is to create three experimental scenarios with different levels of feature redundancy.

Step 1 — Reference sensor R
Criteria: missing_frac < 0.05, then highest standard deviation.
Rationale: R must be reliable (low missing data) and informative (high variance gives a meaningful binary split).
Step 2 — Primary feature A
Criteria: missing_frac < 0.1, ρ(A, R) ≥ 0.85, then highest ρ(A, R).
Rationale: A is the single best feature for predicting the label derived from R. High correlation with R ensures good baseline classification with d=1.
Step 3a — Bhigh (high-correlation scenario)
Criteria: ρ(B, A) ≥ 0.95 and ρ(B, R) ≥ 0.8.
Rationale: B is nearly redundant with A. Adding it should help little asymptotically but hurt at small n (curse of dimensionality) — producing a late crossover.
Step 3b — Bmid (mid-correlation scenario)
Criteria: 0.5 ≤ ρ(B, R) ≤ 0.8 and ρ(B, A) ≤ 0.7.
Rationale: B has moderate correlation — not fully redundant, not fully complementary. This is the regime where double crossover may occur: d=2 starts worse (small n), becomes better (mid n), then the advantage saturates or reverses.
Step 3c — Blow (low-correlation scenario)
Criteria: ρ(B, R) ≤ 0.4 and ρ(B, A) ≤ 0.5.
Rationale: B provides genuinely new information. Adding it should help quickly, producing an early crossover at small n.
